Two Pointers

👆👆 Two Pointers là gì?

🎯
Dùng 2 con trỏ di chuyển trên mảng đã sort
Giảm từ O(n²) xuống O(n)!
# Bài toán: Tìm 2 số có tổng = S trong mảng đã sort
a = [1, 3, 5, 7, 9, 11]
S = 12
left, right = 0, len(a) - 1
while left < right:
    total = a[left] + a[right]
    if total == S:
        print(a[left], a[right])  # 1 11, 3 9, 5 7
        left += 1; right -= 1
    elif total < S: left += 1
    else: right -= 1

🔄 Pattern: Đoạn con liên tiếp

# Tìm đoạn con dài nhất có tổng ≤ S
def max_subarray_len(a, S):
    left = 0
    current_sum = 0
    max_len = 0
    for right in range(len(a)):
        current_sum += a[right]
        while current_sum > S:
            current_sum -= a[left]
            left += 1
        max_len = max(max_len, right - left + 1)
    return max_len
✅ Sliding window — biến thể phổ biến của Two Pointers!

🏆 Bài CP: Xóa phần tử trùng

# Cho mảng sorted, giữ lại mỗi giá trị 1 lần
def unique(a):
    if not a: return []
    result = [a[0]]
    for i in range(1, len(a)):
        if a[i] != a[i-1]:
            result.append(a[i])
    return result

a = [1, 1, 2, 2, 2, 3, 5, 5]
print(unique(a))  # [1, 2, 3, 5]

💪 Bài tập

Bài 1: Tổng 2 số = S ⭐⭐ CP
Mảng N số, tìm cặp có tổng = S (sort + two pointers)
Bài 2: Đoạn con tổng ≤ S ⭐⭐ CP
Tìm đoạn con liên tiếp dài nhất có tổng ≤ S
Bài 3: Merge 2 mảng sorted ⭐⭐
Cho 2 mảng đã sort, gộp thành 1 mảng sorted
Bài 4: Số phần tử khác nhau ⭐⭐⭐ CP
Đếm số phần tử khác nhau trong mảng (sort + two pointers)
