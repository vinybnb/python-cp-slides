🚶 Queue (Hàng đợi) — FIFO

from collections import deque

q = deque()
q.append(1)      # enqueue: [1]
q.append(2)      # enqueue: [1, 2]
q.append(3)      # enqueue: [1, 2, 3]
front = q[0]     # peek: 1
q.popleft()      # dequeue: [2, 3]
📌 deque hỗ trợ thêm/xóa 2 đầu đều O(1)!

🔄 Deque — Double-ended Queue

from collections import deque
d = deque()
d.append(1)       # phải: [1]
d.appendleft(0)   # trái: [0, 1]
d.append(2)       # phải: [0, 1, 2]
d.pop()           # xóa phải: [0, 1]
d.popleft()       # xóa trái: [1]
💡 Dùng cho Sliding Window Maximum — bài CP phổ biến!

🎯 Set — Tập hợp O(1)

s = set()
s.add(5)          # {5}
s.add(3)          # {3, 5}
s.add(5)          # {3, 5} — không trùng!
print(3 in s)     # True — O(1)!
s.remove(3)       # {5}

# Ứng dụng: Đếm phần tử khác nhau
a = [1, 2, 2, 3, 3, 3]
print(len(set(a)))  # 3

📖 Dictionary — Từ điển O(1)

# Đếm tần suất — bài CP siêu phổ biến!
a = [1, 2, 2, 3, 3, 3]
freq = {}
for x in a:
    freq[x] = freq.get(x, 0) + 1
print(freq)  # {1: 1, 2: 2, 3: 3}

# Hoặc dùng Counter
from collections import Counter
freq = Counter(a)
print(freq.most_common(1))  # [(3, 3)]

🏆 Bài CP: Two Sum (dùng Dict)

# Tìm 2 số có tổng = S — O(n) với dict!
def two_sum(a, S):
    seen = {}
    for i, x in enumerate(a):
        complement = S - x
        if complement in seen:
            return [seen[complement], i]
        seen[x] = i
    return [-1, -1]

a = [2, 7, 11, 15]
print(two_sum(a, 9))  # [0, 1] (2 + 7 = 9)

💪 Bài tập

Bài 1: Đếm phần tử khác nhau ⭐

Bài 2: Tần suất ký tự ⭐⭐
