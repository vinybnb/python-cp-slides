🧠 DP là gì?

💡

🐇 Fibonacci: Đệ quy → DP

❌ Đệ quy: O(2ⁿ)

def fib(n):
    if n <= 1: return n
    return fib(n-1) + fib(n-2)
# fib(40) → chạy mãi!

✅ DP: O(n)

def fib(n):
    dp = [0] * (n + 1)
    dp[1] = 1
    for i in range(2, n + 1):
        dp[i] = dp[i-1] + dp[i-2]
    return dp[n]
# fib(1000000) → instant!

🪜 Bài kinh điển: Leo cầu thang

Có N bậc thang. Mỗi bước leo 1 hoặc 2 bậc. Có bao nhiêu cách?
# dp[i] = số cách leo đến bậc i
# dp[i] = dp[i-1] + dp[i-2]  (leo 1 bậc + leo 2 bậc)

def climb(n):
    if n <= 2: return n
    dp = [0] * (n + 1)
    dp[1] = 1
    dp[2] = 2
    for i in range(3, n + 1):
        dp[i] = dp[i-1] + dp[i-2]
    return dp[n]

print(climb(5))  # 8 cách

💰 Đổi tiền xu (DP)

Cho mệnh giá coins[]. Đổi amount đồng với ít xu nhất.
def coin_change(coins, amount):
    dp = [float('inf')] * (amount + 1)
    dp[0] = 0
    for i in range(1, amount + 1):
        for c in coins:
            if c <= i and dp[i - c] + 1 < dp[i]:
                dp[i] = dp[i - c] + 1
    return dp[amount] if dp[amount] != float('inf') else -1

print(coin_change([1, 3, 4], 6))  # 2 (3+3)
⚠️ Greedy cho [1,3,4] với 6 → 4+1+1=3 xu. DP cho 3+3=2 xu. DP đúng hơn!

📝 Template DP

1️⃣Định nghĩa trạng thái: dp[i] = gì?
2️⃣Công thức chuyển: dp[i] tính từ dp[j] nào?
3️⃣Base case: dp[0] = ?, dp[1] = ?
4️⃣Kết quả: dp[n] hay max(dp)?
