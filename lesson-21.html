Graph — Breadth-First Search 🕸️

🕸️ Đồ thị là gì?

🗺️
Tập đỉnh (nodes) + Tập cạnh (edges) nối các đỉnh
Bản đồ, mạng xã hội, mê cung... đều là đồ thị!
# Biểu diễn bằng danh sách kề (adjacency list)
n, m = map(int, input().split())  # n đỉnh, m cạnh
adj = [[] for _ in range(n + 1)]
for _ in range(m):
    u, v = map(int, input().split())
    adj[u].append(v)
    adj[v].append(u)  # Đồ thị vô hướng

🌊 BFS — Duyệt theo chiều rộng

from collections import deque

def bfs(start, adj, n):
    visited = [False] * (n + 1)
    dist = [-1] * (n + 1)
    queue = deque([start])
    visited[start] = True
    dist[start] = 0
    while queue:
        u = queue.popleft()
        for v in adj[u]:
            if not visited[v]:
                visited[v] = True
                dist[v] = dist[u] + 1
                queue.append(v)
    return dist
✅ BFS tìm đường đi ngắn nhất trên đồ thị không trọng số!
