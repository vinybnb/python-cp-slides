🎒 0/1 Knapsack

N vật, mỗi vật có trọng lượng w[i] và giá trị v[i]. Túi chứa W kg. Chọn để tổng giá trị lớn nhất (mỗi vật chỉ chọn 1 lần).
def knapsack(W, weights, values):
    n = len(weights)
    dp = [[0] * (W + 1) for _ in range(n + 1)]
    for i in range(1, n + 1):
        for w in range(W + 1):
            dp[i][w] = dp[i-1][w]  # Không chọn vật i
            if weights[i-1] <= w:
                dp[i][w] = max(dp[i][w],
                    dp[i-1][w - weights[i-1]] + values[i-1])
    return dp[n][W]

W = 7
wt = [1, 3, 4, 5]
val = [1, 4, 5, 7]
print(knapsack(W, wt, val))  # 9 (vật 2+3: 3+4=7kg, 4+5=9)

💾 Tối ưu bộ nhớ: 1D

# Thay vì dp[n][W] → chỉ cần dp[W]
def knapsack_1d(W, weights, values):
    dp = [0] * (W + 1)
    for i in range(len(weights)):
        for w in range(W, weights[i] - 1, -1):  # Duyệt ngược!
            dp[w] = max(dp[w], dp[w - weights[i]] + values[i])
    return dp[W]
⚠️ Duyệt ngược w để mỗi vật chỉ chọn 1 lần!

♾️ Unbounded Knapsack

Mỗi vật có thể chọn nhiều lần
def unbounded_knapsack(W, weights, values):
    dp = [0] * (W + 1)
    for w in range(1, W + 1):
        for i in range(len(weights)):
            if weights[i] <= w:
                dp[w] = max(dp[w], dp[w - weights[i]] + values[i])
    return dp[W]
💡 Khác biệt: duyệt xuôi w → cho phép chọn lại
