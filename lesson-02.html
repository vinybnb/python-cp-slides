if / elif / else — Toán tử so sánh & logic 🧠

🔄 Ôn bài buổi 1 — Quiz nhanh!

Câu 1: print(10 // 3) in ra gì?
→ Đáp án: 3 (chia lấy phần nguyên)
Câu 2: input() trả về kiểu dữ liệu gì?
→ Đáp án: str (luôn luôn là chuỗi!)
Câu 3: 17 % 5 bằng bao nhiêu?
→ Đáp án: 2 (17 chia 5 dư 2)

⚖️ Toán tử so sánh

So sánh 2 giá trị → trả về True hoặc False

Toán tửÝ nghĩaVí dụKết quả
==Bằng5 == 5True
!=Khác5 != 3True
<Nhỏ hơn3 < 7True
>Lớn hơn3 > 7False
<=Nhỏ hơn hoặc bằng5 <= 5True
>=Lớn hơn hoặc bằng3 >= 7False
⚠️ = là gán giá trị, == là so sánh. Đừng nhầm!

🔀 Câu lệnh if

Nếu điều kiện đúng → thực hiện khối lệnh bên trong

tuoi = int(input("Nhập tuổi: "))

if tuoi >= 18:
    print("Bạn đã đủ tuổi trưởng thành! 🎉")
    print("Bạn có thể đi bầu cử")
📌 Lưu ý: Sau if phải có dấu : và khối lệnh bên trong phải thụt vào 4 dấu cách (indent)

🔀 if ... else

Nếu đúng → làm A, Nếu sai → làm B

Điều kiện?
↙️ ↘️
True → Khối if False → Khối else 
so = int(input())

if so % 2 == 0:
    print("Số chẵn")
else:
    print("Số lẻ")

🔀 if ... elif ... else

Khi có nhiều hơn 2 trường hợp

diem = float(input("Nhập điểm: "))

if diem >= 9:
    print("Xuất sắc 🌟")
elif diem >= 8:
    print("Giỏi 👏")
elif diem >= 6.5:
    print("Khá 👍")
elif diem >= 5:
    print("Trung bình 😐")
else:
    print("Yếu — Cần cố gắng hơn! 💪")
📌 Python kiểm tra từ trên xuống, gặp điều kiện đúng đầu tiên thì dừng

🧠 Toán tử logic: and, or, not

Toán tửÝ nghĩaVí dụKết quả
andCả hai phải đúngTrue and FalseFalse
orChỉ cần một đúngTrue or FalseTrue
notĐảo ngượcnot TrueFalse
 
# Kiểm tra số trong khoảng [1, 100]
n = int(input())
if n >= 1 and n <= 100:
    print("Hợp lệ")

# Cách viết ngắn hơn trong Python:
if 1 <= n <= 100:
    print("Hợp lệ")

🏆 Bài CP: Kiểm tra năm nhuận

📋 Đề bài: Nhập năm Y, kiểm tra có phải năm nhuận không.
Năm nhuận: chia hết cho 4 VÀ (không chia hết cho 100 HOẶC chia hết cho 400) 
y = int(input())

if y % 4 == 0 and (y % 100 != 0 or y % 400 == 0):
    print("YES")
else:
    print("NO")

Test thử:

InputOutputGiải thích
2024YES2024 % 4 == 0, 2024 % 100 != 0
1900NO1900 % 100 == 0, 1900 % 400 != 0
2000YES2000 % 400 == 0

💡 Mẹo CP: Edge Cases

🔍
Edge Case = Trường hợp biên / đặc biệt
Những giá trị "kỳ lạ" mà code dễ sai nếu không xét
⚠️Số 0 — Chẵn hay lẻ? Dương hay âm?
⚠️Số âm — Code có xử lý đúng không?
⚠️Giá trị rất lớn — 10⁹, 10¹⁸?
⚠️Giá trị bằng nhau — a == b thì sao?
✅ Quy tắc vàng: Luôn thử với giá trị nhỏ nhất, lớn nhất, bằng 0, bằng nhau, và âm!
