📦 Biến (Variable)

Biến giống như hộp chứa dữ liệu — đặt tên để dùng lại

# Tạo biến
ten = "Minh"
tuoi = 12
diem_toan = 9.5

# Sử dụng biến
print("Xin chào, tôi là", ten)
print("Năm nay tôi", tuoi, "tuổi")
print("Điểm Toán:", diem_toan)
📌 Quy tắc đặt tên biến: Không bắt đầu bằng số, không có dấu cách, nên dùng tiếng Anh

📊 Kiểu dữ liệu cơ bản

KiểuÝ nghĩaVí dụ

floatSố thực (có phần thập phân)3.14, -2.5

⌨️ Nhập dữ liệu: input()

# Nhập tên
ten = input("Nhập tên của bạn: ")
print("Xin chào", ten)

# Nhập số (phải chuyển đổi!)
tuoi = int(input("Nhập tuổi: "))
print("Năm sau bạn", tuoi + 1, "tuổi")
⚠️ Lưu ý quan trọng: input() luôn trả về str.
Muốn tính toán → phải chuyển bằng int() hoặc float()

🏆 Format bài thi CP

Trong CP, bài toán luôn có dạng: Đọc Input → Xử lý → In Output

📥 Input → ⚙️ Xử lý → 📤 Output

📥 Ví dụ Input

3 5

📤 Output mong đợi

8
# Lời giải: Tính tổng 2 số
a, b = map(int, input().split())
print(a + b)

🪄 Giải mã "câu thần chú"

a, b = map(int, input().split())
1️⃣input() → đọc dòng: "3 5"
2️⃣.split() → tách bởi dấu cách: ["3", "5"]
3️⃣map(int, ...) → chuyển mỗi phần tử thành số: 3, 5
4️⃣a, b = ... → gán: a = 3, b = 5
✨ Mẹo: Học thuộc pattern này — dùng trong hầu hết mọi bài CP!
