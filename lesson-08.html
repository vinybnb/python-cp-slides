🫧 Bubble Sort

So sánh 2 phần tử liền kề, đổi chỗ nếu sai thứ tự

def bubble_sort(a):
    n = len(a)
    for i in range(n):
        for j in range(n - 1 - i):
            if a[j] > a[j+1]:
                a[j], a[j+1] = a[j+1], a[j]
    return a
⏱️ Độ phức tạp: O(n²) — chậm với dữ liệu lớn!

🎯 Selection Sort

Tìm min, đặt vào đầu, lặp lại

def selection_sort(a):
    n = len(a)
    for i in range(n):
        min_idx = i
        for j in range(i+1, n):
            if a[j] < a[min_idx]:
                min_idx = j
        a[i], a[min_idx] = a[min_idx], a[i]
    return a
📌 Cũng O(n²) — nhưng ít swap hơn Bubble Sort

⚡ Python built-in sort

Trong CP, dùng sort() có sẵn — O(n log n)

a = [5, 2, 8, 1, 9]
a.sort()              # Sắp xếp tăng dần tại chỗ
a.sort(reverse=True)  # Giảm dần

b = sorted(a)         # Tạo list mới đã sort

# Sort theo tiêu chí tùy chỉnh
pairs = [(3, 'c'), (1, 'a'), (2, 'b')]
pairs.sort(key=lambda x: x[0])  # Sort theo phần tử đầu
✅ Trong CP luôn dùng .sort() — nhanh & đáng tin cậy!

🏆 Bài CP: Sort tùy chỉnh

# Sắp xếp học sinh theo điểm giảm dần,
# nếu bằng điểm thì theo tên tăng dần
students = []
n = int(input())
for _ in range(n):
    name, score = input().split()
    students.append((name, int(score)))

students.sort(key=lambda x: (-x[1], x[0]))
for name, score in students:
    print(name, score)
💡 Dấu - trước số = đảo thứ tự sort cho trường đó
