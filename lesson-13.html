Buổi 13

Prefix Sum

Tổng tiền tố — Truy vấn đoạn O(1) 📐

🤔 Bài toán

Cho mảng N số, Q truy vấn. Mỗi truy vấn: tính tổng đoạn [L, R]

❌ Brute force: O(N×Q)

for _ in range(Q):
    l, r = map(int, input().split())
    print(sum(a[l:r+1]))

✅ Prefix Sum: O(N+Q)

pre = [0] * (n+1)
for i in range(n):
    pre[i+1] = pre[i] + a[i]
# Tổng [l,r] = pre[r+1] - pre[l]

📐 Cách xây dựng Prefix Sum

a = [3, 1, 4, 1, 5, 9]
#     0  1  2  3  4  5

# Xây mảng prefix sum
pre = [0]  # pre[0] = 0
for x in a:
    pre.append(pre[-1] + x)
# pre = [0, 3, 4, 8, 9, 14, 23]

# Tổng đoạn [l, r] (0-indexed)
def range_sum(l, r):
    return pre[r+1] - pre[l]

print(range_sum(1, 3))  # a[1]+a[2]+a[3] = 1+4+1 = 6

🏆 Bài CP: Đoạn con tổng lớn nhất

# Kadane's Algorithm — O(n)
def max_subarray_sum(a):
    max_sum = a[0]
    current = a[0]
    for i in range(1, len(a)):
        current = max(a[i], current + a[i])
        max_sum = max(max_sum, current)
    return max_sum

a = [-2, 1, -3, 4, -1, 2, 1, -5, 4]
print(max_subarray_sum(a))  # 6 (đoạn [4,-1,2,1])

🔢 Mảng hiệu (Difference Array)

# Cộng giá trị v vào đoạn [l,r] — nhiều truy vấn
n = 10
diff = [0] * (n + 1)

# Cộng 5 vào đoạn [2, 6]
diff[2] += 5
diff[7] -= 5

# Khôi phục mảng gốc
a = [0] * n
a[0] = diff[0]
for i in range(1, n):
    a[i] = a[i-1] + diff[i]
💡 Prefix Sum ngược lại: cập nhật đoạn O(1), truy vấn O(n)

💪 Bài tập

Bài 1: Range Sum Query ⭐⭐ CP
N số, Q truy vấn [L,R] → tổng đoạn
Bài 2: Đoạn con tổng = S ⭐⭐ CP
Đếm số đoạn con có tổng bằng S
Bài 3: Max subarray sum ⭐⭐⭐ CP
Tìm đoạn con có tổng lớn nhất (Kadane)
