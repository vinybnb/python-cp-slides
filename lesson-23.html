Đường đi ngắn nhất

Dijkstra & Floyd-Warshall 🛣️

🛣️ Dijkstra

Tìm đường đi ngắn nhất từ 1 đỉnh đến tất cả đỉnh còn lại (trọng số ≥ 0)
import heapq

def dijkstra(start, adj, n):
    dist = [float('inf')] * (n + 1)
    dist[start] = 0
    pq = [(0, start)]  # (khoảng cách, đỉnh)
    while pq:
        d, u = heapq.heappop(pq)
        if d > dist[u]: continue
        for v, w in adj[u]:
            if dist[u] + w < dist[v]:
                dist[v] = dist[u] + w
                heapq.heappush(pq, (dist[v], v))
    return dist
⏱️ O((V + E) log V) với priority queue

🏆 Ví dụ Dijkstra

# Đọc input: n đỉnh, m cạnh có trọng số
n, m = map(int, input().split())
adj = [[] for _ in range(n + 1)]
for _ in range(m):
    u, v, w = map(int, input().split())
    adj[u].append((v, w))
    adj[v].append((u, w))

dist = dijkstra(1, adj, n)
for i in range(1, n + 1):
    print(f"Đỉnh 1 → {i}: {dist[i]}")

🔄 Floyd-Warshall

Tìm đường ngắn nhất giữa mọi cặp đỉnh — O(n³)
def floyd(n, adj_matrix):
    dist = [row[:] for row in adj_matrix]
    for k in range(n):
        for i in range(n):
            for j in range(n):
                if dist[i][k] + dist[k][j] < dist[i][j]:
                    dist[i][j] = dist[i][k] + dist[k][j]
    return dist
📌 Dùng khi N ≤ 500 và cần khoảng cách mọi cặp

📊 Khi nào dùng gì?
