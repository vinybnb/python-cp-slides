🔁 Vòng lặp for

Lặp lại một hành động nhiều lần

# In "Hello" 5 lần
for i in range(5):
    print("Hello", i)

# Kết quả:
# Hello 0
# Hello 1
# Hello 2
# Hello 3
# Hello 4
📌 range(5) tạo dãy số: 0, 1, 2, 3, 4 (5 số, bắt đầu từ 0)

📏 range() chi tiết

Cách dùngTạo dãyÝ nghĩa
range(5)0, 1, 2, 3, 4Từ 0 đến 4
range(1, 6)1, 2, 3, 4, 5Từ 1 đến 5
range(0, 10, 2)0, 2, 4, 6, 8Từ 0 đến 8, bước 2
range(5, 0, -1)5, 4, 3, 2, 1Đếm ngược!
 
# Tính tổng 1 + 2 + ... + N
n = int(input())
tong = 0
for i in range(1, n + 1):
    tong += i      # tong = tong + i
print(tong)

🔄 Vòng lặp while

Lặp khi điều kiện còn đúng

# Đếm ngược phóng tên lửa!
dem = 5
while dem > 0:
    print(dem, "...")
    dem -= 1
print("🚀 Phóng!")

⏹️ break & continue

break — Dừng lặp

# Tìm số đầu tiên chia hết cho 7
for i in range(1, 100):
    if i % 7 == 0:
        print("Tìm thấy:", i)
        break
# → Tìm thấy: 7

continue — Bỏ qua

# In số từ 1-10, bỏ qua số chẵn
for i in range(1, 11):
    if i % 2 == 0:
        continue
    print(i)
# → 1 3 5 7 9

💪 Tư duy Brute Force

🔨
Brute Force = "Vét cạn"
Thử TẤT CẢ khả năng, kiểm tra cái nào thỏa mãn
# Đếm số ước của N
n = int(input())
dem = 0
for i in range(1, n + 1):
    if n % i == 0:     # i là ước của n
        dem += 1
print(dem)
✅ Brute force là cách giải đơn giản nhất — luôn thử cách này trước!

🏆 Bài CP kinh điển: Số nguyên tố

📋 Số nguyên tố là số > 1 và chỉ chia hết cho 1 và chính nó
Ví dụ: 2, 3, 5, 7, 11, 13, 17, 19, 23, ... 
n = int(input())

if n < 2:
    print("NO")
else:
    la_nguyen_to = True
    for i in range(2, n):
        if n % i == 0:
            la_nguyen_to = False
            break
    
    if la_nguyen_to:
        print("YES")
    else:
        print("NO")
💡 Tối ưu: Chỉ cần kiểm tra đến √N → range(2, int(n**0.5) + 1)

🔁🔁 Vòng lặp lồng nhau

# In bảng cửu chương
for i in range(2, 10):
    print(f"--- Bảng {i} ---")
    for j in range(1, 11):
        print(f"{i} x {j} = {i * j}")
    print()  # dòng trống
📌 f"..." là f-string — cách format chuỗi tiện lợi trong Python.
Đặt biến trong {} để in giá trị

💪 Bài tập thực hành

Bài 1: Bảng cửu chương ⭐
Nhập N, in bảng cửu chương của N
Bài 2: Tổng 1→N ⭐
Nhập N, tính 1 + 2 + ... + N
Bài 3: N! Giai thừa ⭐⭐
Nhập N, tính N! = 1 × 2 × 3 × ... × N
Bài 4: Đếm ước ⭐⭐ CP
Nhập N, đếm và in ra tất cả ước của N
Bài 5: Kiểm tra nguyên tố ⭐⭐⭐ CP
Nhập N, in "YES" nếu N là nguyên tố, "NO" nếu không
Bài 6: Liệt kê nguyên tố ⭐⭐⭐ CP
Nhập N, in tất cả số nguyên tố ≤ N
