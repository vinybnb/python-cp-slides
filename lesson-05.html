🔤 String (Chuỗi) cơ bản

String = dãy các ký tự, có thể duyệt như list

s = "Hello CP"

# Truy cập ký tự
print(s[0])     # H
print(s[-1])    # P
print(len(s))   # 8

# Duyệt từng ký tự
for ch in s:
    print(ch, end=" ")
# H e l l o   C P
📌 String trong Python là immutable — không thể sửa từng ký tự!

🔢 ord() và chr()

Mỗi ký tự có một mã số ASCII

# ord() → Ký tự thành số
print(ord('A'))   # 65
print(ord('a'))   # 97
print(ord('0'))   # 48

# chr() → Số thành ký tự
print(chr(65))    # A
print(chr(97))    # a
💡 Mẹo quan trọng:
• 'A' đến 'Z': mã 65–90
• 'a' đến 'z': mã 97–122
• '0' đến '9': mã 48–57

🏆 Bài CP: Palindrome

📋 Palindrome = chuỗi đọc xuôi giống đọc ngược
VD: "madam", "12321", "abcba" 
s = input()

# Cách 1: So sánh với chuỗi đảo
if s == s[::-1]:
    print("YES")
else:
    print("NO")

# Cách 2: So sánh 2 đầu (tư duy CP)
ok = True
for i in range(len(s) // 2):
    if s[i] != s[len(s) - 1 - i]:
        ok = False
        break
print("YES" if ok else "NO")
💡 s[::-1] là cách đảo chuỗi ngắn nhất trong Python!

🔐 Bài CP: Mã hóa Caesar

📋 Dịch mỗi chữ cái đi K vị trí trong bảng chữ cái.
VD: K=3 → A→D, B→E, ..., X→A, Y→B, Z→C 
s = input()
k = int(input())
result = ""

for ch in s:
    if ch.isalpha():
        base = ord('A') if ch.isupper() else ord('a')
        # Dịch k vị trí, quay vòng bằng % 26
        new_ch = chr((ord(ch) - base + k) % 26 + base)
        result += new_ch
    else:
        result += ch  # Giữ nguyên ký tự không phải chữ

print(result)
📌 % 26 giúp quay vòng: Z + 1 → A

📋 Bài A: Tổng chữ số 100 điểm

Đề bài: Nhập số nguyên dương N (1 ≤ N ≤ 10⁹). In ra tổng các chữ số của N.

Ví dụ:
Input: 1234 → Output: 10
Input: 999 → Output: 27

Gợi ý: Dùng vòng while + phép chia dư % 10

📋 Bài B: Palindrome 100 điểm

Đề bài: Nhập chuỗi S (chỉ chứa chữ cái thường, 1 ≤ |S| ≤ 10⁵). Kiểm tra S có phải palindrome không.

Ví dụ:
Input: madam → Output: YES
Input: hello → Output: NO

Gợi ý: So sánh chuỗi với chuỗi đảo ngược

📋 Bài C: Đếm nguyên tố 150 điểm

Đề bài: Nhập 2 số A, B (2 ≤ A ≤ B ≤ 10⁶). Đếm số nguyên tố trong đoạn [A, B].

Ví dụ:
Input: 2 10 → Output: 4 (các số: 2, 3, 5, 7)

Gợi ý: Viết hàm kiểm tra nguyên tố, duyệt từ A đến B

📋 Bài D: Số lớn thứ 2 150 điểm

Đề bài: Nhập mảng N số (2 ≤ N ≤ 10⁵). Tìm số lớn thứ 2 (giá trị khác max).

Ví dụ:
Input: 5
3 1 4 4 2 → Output: 3
(Max = 4, số lớn thứ 2 khác max = 3)

Gợi ý: Tìm max trước, rồi tìm max trong các phần tử ≠ max

📋 Bài E: Caesar Cipher 200 điểm

Đề bài: Nhập chuỗi S và số K. Mã hóa Caesar: dịch mỗi chữ cái đi K vị trí, giữ nguyên ký tự khác.

Ví dụ:
Input: Hello, World!
3
Output: Khoor, Zruog!

Gợi ý: Dùng ord(), chr(), % 26 để quay vòng

def is_prime(n):
    if n < 2:
        return False
    for i in range(2, int(n**0.5) + 1):
        if n % i == 0:
            return False
    return True

a, b = map(int, input().split())
dem = 0
for x in range(a, b + 1):
    if is_prime(x):
        dem += 1
print(dem)

✅ Lời giải Bài D — Số lớn thứ 2

n = int(input())
a = list(map(int, input().split()))

max_val = max(a)
second = -float('inf')  # Âm vô cùng

for x in a:
    if x != max_val and x > second:
        second = x

print(second)
